# 🧙‍♂️ Harry Potter API Website

Um site interativo inspirado no universo de Harry Potter, desenvolvido com **HTML**, **CSS** e **JavaScript**, consumindo APIs públicas e trazendo uma experiência mágica ao usuário ✨

---

## 🌐 Acesse o projeto

👉 [Harry Potter API](https://carolfantinate.github.io/harry-potter-api/)  

---

## 📌 Sobre o projeto

O projeto foi criado com o objetivo de praticar consumo de API, manipulação do DOM e criação de interfaces modernas e interativas.

Aqui você pode explorar o mundo mágico através de diferentes seções:

- 🏠 Casas de Hogwarts  
- 👤 Personagens  
- 🪄 Feitiços  
- 🎯 Quiz: descubra sua casa  

---

## ✨ Funcionalidades

- 🔎 **Busca em tempo real** de personagens e feitiços  
- 🎯 **Filtro por casas** (Grifinória, Sonserina, Corvinal e Lufa-Lufa)  
- 🧠 **Quiz interativo** que define sua casa  
- 🔗 Integração entre páginas (ex: clicar na casa → ver personagens filtrados)  
- 🪄 **Cursor mágico com varinha + efeitos visuais**  
- 🎨 Animações e efeitos (hover, partículas, transições)

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3 (animações, layout responsivo)  
- JavaScript (DOM, eventos, fetch API)  
- APIs externas:
  - https://potterhead-api.vercel.app/api/characters  
  - https://potterapi-fedeperin.vercel.app/pt/spells
