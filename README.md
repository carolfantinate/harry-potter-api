# 🧙‍♂️ Harry Potter API Website

Um projeto simples de site desenvolvido com **HTML**, **CSS** e **JavaScript**, consumindo uma API do universo de Harry Potter.

## 🌐 Visualização

👉 [Clique aqui para acessar o site](https://carolfantinate.github.io/harry-potter-api/)

## 📌 Sobre o projeto

Este site exibe informações sobre o mundo mágico, incluindo:

- 🏠 Casas de Hogwarts  
- 👤 Personagens  

Os dados são obtidos através de uma API pública.

## 🚧 Status do projeto

⚠️ Em desenvolvimento

Novas funcionalidades estão sendo planejadas, como:

- ✨ Listagem de feitiços  
- 🔎 Sistema de busca  
- 🎯 Filtros para personagens e casas  
- 📊 Mais detalhes sobre o universo  

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- API externa
